# Aplicação de Galeria de Fotos

## Visão Geral

Aplicação web que exibe uma galeria de fotos da fauna brasileira, permitindo a visualização, filtragem e pesquisa de imagens por nome. O projeto foi desenvolvido com foco em responsividade, acessibilidade e experiência do usuário.

---

## Tema

Fauna Brasileira

---

## Funcionalidades

- Exibição de galeria de fotos em layout grid responsivo
- Busca em tempo real por nome da foto com debounce otimizado
- Efeito visual ao passar o mouse sobre as imagens
- Visualização ampliada da foto em modal ao clicar
- Navegação por teclado e suporte a tecla ESC para fechar modal
- Mensagem informativa quando nenhuma foto é encontrada
- Layout totalmente responsivo para diferentes tamanhos de tela
- Lazy loading de imagens para melhor performance

---

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
  - Flexbox e Grid Layout
  - Variáveis CSS
  - Media Queries
- **JavaScript**
  - Manipulação do DOM
  - Event Listeners
  - Debounce para otimização de busca

## Como Executar o Projeto

### Pré-requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Editor de código (opcional, caso deseje modificar)

### Passo a passo

1. **Clonar o repositório**

```bash
git clone https://github.com/seu-usuario/galeria-fauna-brasileira.git
```

2. **Acessar a pasta do projeto**

```bash
cd galeria-fauna-brasileira
```

3. **Executar a aplicação**

Abra o arquivo `index.html` diretamente no navegador ou utilize uma extensão como Live Server no VS Code.
